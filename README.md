## Deskripsi

Proyek ini merupakan implementasi aplikasi Sistem Perpustakaan menggunakan Laravel dengan konsep MVC (Model-View-Controller). Fitur yang dibuat meliputi pengelolaan data anggota dan kategori buku, routing, controller, blade template, migration, model, dan seeder database.

## Fitur

- Daftar dan detail anggota perpustakaan
- Daftar, detail, dan pencarian kategori buku
- Layout Blade dengan Bootstrap 5
- Migration tabel kategori
- Model Eloquent Kategori
- Seeder data kategori

## Teknologi

- Laravel
- PHP
- MySQL
- Bootstrap 5
